如何使用

如何使用

查收邮件

付款后，请先查收你的邮件。如果有内容缺失，请直接回复邮件，我们会尽快补发。

我们收到付款后，会给你回复一个邮件。邮件中会包含：

  1. 二维码
  2. ss信息

请收到邮件后将二维码保存到手机相册或电脑中。

下面请按照你设备的平台来安装对应的客户端。

安装客户端

安卓系统

请安装Shadowsocks for Android。

iOS系统

请在电脑安装PP助手，安装成功后启动pp助手。使用usb线将手机连接到电脑，电脑端的pp助手会自动在手机上安装手机版pp助手。

在电脑上下载Shadowrocket for iOS，在电脑端选择应用安装，选择上述安装包，安装到手机上。

手机上出现shadowrocket（图标是一个小火箭）图标即成功。

windows系统

直接下载Shadowsocks for Windows安装即可。

macOS系统

直接下载ShadowsocksX-NG for macOS安装即可。

安装成功后

安装成功后，手机端点击右上角的添加，桌面端在右键菜单中点击添加配置。或是从相册选择二维码，或是选择手动添加均可。手动添加则需要按照邮件中提供的ss地址、密码等信息手动填写。

客户端更新

上面教程的客户端版本截止2019年10月09日，在你阅读此教程时，这些客户端可能已经不是最新了。

如果你不是普通用户，想要自己的shadowsocks客户端保持最新，这里整合了各平台shadowsocks的release页面地址，你可以前往安装最新客户端。

看不懂教程

你可以试着发送邮件请求手把手教学，如果我们在闲，会很热心地帮助你完成安装的。
